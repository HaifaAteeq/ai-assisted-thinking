## Quick Reference: Detection Signals

### SHALLOW (AI Should Catch)

| Signal | Example |
|--------|---------|
| Vague | "It's used for data" |
| No example | Abstract explanation only |
| Copied words | Same phrases AI used |
| List only | "A, B, C, and D" |
| WHAT only | "It does X" (no why) |
| Mixed | Several concepts jumbled |

### DEEP (AI Should Accept)

| Signal | Example |
|--------|---------|
| Specific | "Like [0.8, 0.2] for email" |
| Own words | Different phrasing |
| Connected | "X causes Y because Z" |
| WHY included | "Because the network needs..." |
| Can apply | "So for this case..." |
| Knows gaps | "I understand X but not Y" |
