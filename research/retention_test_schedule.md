## Test Rules

### How to Take Tests

1. **No looking at notes** - Answer from memory only
2. **Write answers first** - Then check
3. **Be honest** - Partial memory counts as partial
4. **Document everything** - Even failures are data

### Scoring

| Score | Meaning |
|-------|---------|
| ✅ Full | Explained correctly with reasoning |
| 🔄 Partial | Some correct, some missing |
| ❌ Lost | Could not recall or incorrect |

## If Retention Drops Below 70%

### Action Plan

1. **Identify weak topics** from test results
2. **Quick review session** (15-30 min) - not full re-learning
3. **Re-test weak topics** in 2-3 days
4. **Adjust future learning** - spend more time on verification

### Review Session Format

```
AI: "Let's do a quick review of [weak topic].

Without me explaining, tell me what you remember about [topic].
Then I'll fill in gaps and we'll verify again."
```
